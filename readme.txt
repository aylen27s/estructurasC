Programa de carga de datos de corredores

Informacion que permite ingresar:
-ID(valor numerico entre 1000 9999
-Nombre y apellido
-Marca de tiempo del corredor
-Distancia recorrida (valor numerico en km)

Al finalizar la carga provee dos listas. La primera muestra todos los datos ingresados
y la segunda ofrece la velocidad asociada a cada corredor.

Opciones para listar la informacion:
- V : ordena por velocidad de manera ascendente.
- N : ordena alfabeticamente de manera ascendete.
